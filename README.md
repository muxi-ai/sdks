# MUXI SDKs

Official SDKs for integrating with the MUXI platform. Each SDK handles auth, transports, idempotency, retries, and streaming with language-idiomatic APIs.

> Looking for the CLI? See [muxi-ai/cli](https://github.com/muxi-ai/cli).
